Data Visualization of NYPD Reported Hate Crimes

I was interested in looking into the hate crimes committed throughout New York. The original data had too much information (14 columns and 1908 rows), so I decided to run a SQL query that would group hate crimes based off the group that was targeted, and then visualize that.
